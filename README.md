# 🐟 Dos Dragones | Landing Page

Landing page desarrollada para **Dos Dragones**, una tienda enfocada en el cuidado responsable de especies y acuarios.

El objetivo del sitio es presentar información clara sobre servicios, productos y contenido educativo, además de facilitar el contacto con los clientes.

## ✨ Características

- Diseño moderno y responsivo
- Secciones informativas (Inicio, Conócenos, Productos, Servicios, FAQ)
- Sistema de navegación fluida (scroll)
- Sección de tutoriales con enlaces a contenido externo
- Cards interactivas y modales para productos
- Optimización visual y estructura enfocada en UX

---

## 🛠️ Tecnologías utilizadas

- Astro
- Tailwind CSS
- JavaScript
- HTML5

---

## 🎯 Objetivo del proyecto

Crear una landing page atractiva y funcional que permita:

- Mostrar información del negocio de forma clara
- Generar confianza en los usuarios
- Facilitar el contacto mediante redes sociales
- Mejorar la presencia digital del negocio
